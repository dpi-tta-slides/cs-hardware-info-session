# Tech Roles and Interviewing

The objective of this course is to provide a foundation for high school teachers to teach electronics. This course bootstraps teachers with hardware knowledge from Ohm’s law to Microcontrollers. Students will get exposure to further studies in Computer Science, Computer Engineering, Electrical Engineering, and other STEM fields. The pedagogical approach is learning by discovery. We will work on projects that will build intuition for the underlying theory.
